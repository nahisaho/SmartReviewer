## 1. 対象文書種別

### 1.1 MVP対象文書

本PoCでは、以下の2種類の文書をレビュー対象とする。

| # | 文書種別 | 説明 | 想定文書量 |
|---|----------|------|------------|
| 1 | **基本設計書** | システムの基本設計を記述した文書。アーキテクチャ、機能設計、データ設計等を含む | 3万〜10万文字 |
| 2 | **全体テスト計画書** | テスト全体の計画を記述した文書。テスト方針、スコープ、スケジュール、体制等を含む | 2万〜5万文字 |

### 1.2 選定理由

| 観点 | 基本設計書 | 全体テスト計画書 |
|------|------------|------------------|
| **重要性** | システム開発の根幹となる成果物 | 品質保証の基盤となる成果物 |
| **標準準拠** | ガイドラインに明確な指針あり | ガイドラインに明確な指針あり |
| **チェック可能性** | 構造化されたチェック項目が定義可能 | 構造化されたチェック項目が定義可能 |
| **評価データ** | サンプル文書の入手可能性が高い | サンプル文書の入手可能性が高い |

### 1.3 参照ガイドライン

| 文書種別 | 主要参照ガイドライン |
|----------|---------------------|
| 基本設計書 | デジタル・ガバメント推進標準ガイドライン 第3編 第6章 |
| 全体テスト計画書 | デジタル・ガバメント推進標準ガイドライン 第3編 第7章 |

## 4. チェック項目のRAG方式マッピング

### 4.1 チェック種別別の分布

| チェック種別 | 基本設計書 | テスト計画書 | 合計 | RAG方式 |
|-------------|-----------|-------------|------|---------|
| 単純チェック | 3項目 | 2項目 | 5項目 | Vector RAG |
| 関係性チェック | 4項目 | 4項目 | 8項目 | Graph RAG |
| 網羅性チェック | 3項目 | 4項目 | 7項目 | Ontology + Graph RAG |

              └─────────────────────┘
                         │
         ┌───────────────┼───────────────┐
         ▼               ▼               ▼
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│ Vector RAG  │  │ Graph RAG   │  │ Hybrid RAG  │
│             │  │             │  │             │
│ BD-C01,C02  │  │ BD-B01,B03  │  │ BD-B02,A01  │
│ BD-A03      │  │ BD-D01,C03  │  │ BD-D02      │

## 5. 評価データセット要件

### 5.1 データセット構成

| 文書種別 | 件数 | 内容 |
|----------|------|------|
| 基本設計書 | 10件 | 過去プロジェクトの設計書（匿名化） |
| 全体テスト計画書 | 10件 | 過去プロジェクトのテスト計画書（匿名化） |

### 5.2 正解データ形式

各評価文書に対して、以下の正解データを作成する：

```json
{
  "document_id": "BD-EVAL-001",
  "document_type": "basic_design",
  "check_results": [
    {
      "check_item_id": "BD-B01",
      "result": "NG",
      "evidence": "第3章の機能設計において、要件REQ-012に対応する設計が見当たらない",
      "location": "第3章 機能設計",
      "severity": "High",
      "suggested_correction": "要件REQ-012に対応する機能設計を追加する"
    }
  ]
}
```

### 5.3 評価指標

| 指標 | 算出方法 | 目標値 |
|------|----------|--------|
| チェック合致率 | (AIの判定 = 正解) / 全判定数 | ≥ 70% |
| 精度 (Precision) | TP / (TP + FP) | ≥ 70% |
| 再現率 (Recall) | TP / (TP + FN) | ≥ 65% |
| F1スコア | 2 × P × R / (P + R) | ≥ 67% |
